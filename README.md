# DSA_Java
Collection of data structure and algorithm questions solved in Java.

## Solved problems
- Ques_1: Check whether array is sorted.
- Ques_2: Merge two sorted arrays.
- Ques_3: Find second largest element in array.
- Ques_4: Sum of all array elements.
- Ques_5: Remove duplicate elements from array.
- Ques_6: Find pair with given sum in array.
- Ques_7: Check whether a string is palindrome.
- Ques_8: Binary search in sorted array.
